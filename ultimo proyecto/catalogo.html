Sensor de Temperatura IoT
  • Modelo: TempSense X1
  • Rango: -20°C a 60°C
  • Conectividad: Wi-Fi

Sensor avanzado que permite monitorear en tiempo real la temperatura del invernadero mediante nuestra plataforma web. Ideal para mantener condiciones óptimas de cultivo.

Controlador de Humedad
  • Modelo: HumidControl Z3
  • Rango: 0% a 100%
  • Precisión: ±2%

Este dispositivo asegura niveles de humedad ideales en el invernadero, ajustando automáticamente según las necesidades específicas de los cultivos.

Sistema de Riego Inteligente
  • Modelo: AquaFlow Pro
  • Capacidad: Hasta 500 m²
  • Control: App Móvil/Web

Un sistema automatizado que optimiza el consumo de agua, garantizando un riego eficiente y sostenible. Configurable a través de la plataforma web o app móvil.
